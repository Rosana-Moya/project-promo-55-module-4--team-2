Ada Sign
Este es el proyecto del Módulo 4 del Equipo 2 de la Promoción Reshma Saujani. Proyectos Molones es una aplicación web desarrollada en JavaScript, HTML, CSS, REACT y NodeJS. Permite publicar un proyecto de programación a través de un formulario, visualizar un listado de proyectos publicados por otras programadoras y eliminar el proyecto subido.

Funcionalidades:
Subir y previsualizar un proyecto a través de un formulario.
Completar campos de información (nombre del proyecto, slogan, repositorio, Demo, Tecnologías utilizadas, Descripción, información de la autora como nombre y trabajo).
Vista previa en tiempo real del proyecto completándose.
Guardado automático en LocalStorage.
Listado de proyectos publicados.
Visualización de un proyecto al detalle.
Botón de eliminar proyecto.

Tecnologías utilizadas:
HTML5 – estructura de la aplicación.
Sass/SCSS – estilos organizados en parciales.
JavaScript (ES6) y REACT – lógica de interacción.
LocalStorage – persistencia de datos en el navegador.
NodeJS - lógica de funcionamiento de la API en Backend.
MySQL Workbench - base de datos para almacenar información.
